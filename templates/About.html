About The Data


What is this data?

This data is an assortment of all reddit comments from r/politics from the month of November in 2016. The data came in the form of a json dump with the comments, upvotes/downvotes, guild status, commenter id and date attatched.


Where did we get this data?

We sourced our data from the top comment from r/datasets, which we happened upon from searching the term "dataset" into the reddit search bar. We downloaded the json dump from academictorrents.com, but ended up using the one month of data hosted on the following mega link:

https://mega.nz/#!ysBWXRqK!yPXLr25PgJi184pbJU3GtnqUY4wG7YvuPpxJjEmnb9A.

The magnet link to the full set of data can be found here

No licensing information was given on this dataset, since it was scraped by an individual, however the author gave explicit permition to use this data for academic use and did not list any restrictions that would limit our use of this data, thus it is our belief that this is a permissable use of the data which is inline with the author specifications. The author that created this data set is reddit user u/Stuck_In_the_Matrix who posted this database in 2015 and updated it in 2017.


Citation and Manipulations

The entire data set contains all publicly available Reddit comments over the history of reddit up through 2015. There are about 1.7 billion comments, however we filtered the data to all the comments in the election month of 2017 due to storage and scope limitations. There are no geographic restrictions on the comments, however we tried to filter all the comments from the r/political subreddit to make our analysis relevant to political discourse. The overview and original post can be found at this link : https://www.reddit.com/r/datasets/comments/3bxlg7/i_have_every_publicly_available_reddit_comment/ There is no citation provided, but the link above should suffice in terms of tracing the datasets technical documentation. Finally, the data is mostly in its unaltered forms, we only removed certain columns (like the subreddit ID) that were homogenous across all comments or not relevant to our analysis.
